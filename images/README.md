# Images 폴더

이 폴더에는 익스텐션에 사용할 이미지 파일들을 저장합니다.

## 필요한 파일

### 필수 파일

- `icon.png` - 메인 아이콘 (128x128 픽셀, PNG 형식)
  - VS Code 마켓플레이스와 익스텐션 상세 페이지에 표시됩니다

### 선택 파일

- `screenshot.png` - README에 사용할 스크린샷
- `activity-icon.svg` - Activity Bar 커스텀 아이콘 (선택사항)

## 아이콘 제작 가이드

1. **크기**: 128x128 픽셀
2. **형식**: PNG (투명 배경 권장)
3. **스타일**:
   - 간단하고 명확한 디자인
   - 작은 크기에서도 잘 보이는 형태
   - VS Code의 디자인 가이드라인 준수

## 아이콘 제작 도구 추천

- [Figma](https://www.figma.com/) - 무료 디자인 도구
- [Canva](https://www.canva.com/) - 간단한 아이콘 제작
- [Icon Generator](https://www.favicon-generator.org/) - 온라인 아이콘 생성기
